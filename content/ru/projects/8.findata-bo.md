## О проекте

Findata BO — production-ready финтех-система в виде Telegram-бота для инвесторов на российском облигационном рынке. Агрегирует данные из 6+ официальных источников (e-disclosure.ru, НРД, АКРА, НРА, Эксперт РА, isin.ru) и доставляет персонализированные уведомления с учётом ISIN-кодов облигаций пользователя.

## Задача

Решить критическую проблему инвестора: оперативное отслеживание десятков компаний-эмитентов на разрозненных площадках. Автоматизировать мониторинг, превратив ручную работу в автоматическую доставку релевантных уведомлений — от событий раскрытия до появления новых облигаций на бирже. С полной монетизацией.

## Архитектура

### Двухпроцессная модель

**Bot Process** — Telegram-взаимодействия через aiogram 3, FSM для диалогов, 6 роутеров.

**Worker Process** — 6 параллельных воркеров через `asyncio.gather()`:
- CompanyWorker (30с) — мониторинг подписок
- BroadcastWorker (30с) — глобальные уведомления по ISIN
- TbankIsinWorker — мониторинг новых облигаций
- SubscriptionWorker (1ч) — управление подписками
- TbankSyncWorker (ночное окно 01:00-05:00) — delta sync портфеля
- PaymentWorker (30с, 24/7) — проверка статусов YooKassa

### Source Registry

Расширяемая архитектура: `NewsSource` (ABC + Generic) с 5 абстрактными методами, `NewsEvent` dataclass, `SourceRegistry` синглтон. Добавление нового источника — реализация ABC-класса и регистрация.

### Двухуровневый Data Access

SQLAlchemy ORM — только для Alembic-миграций. asyncpg raw queries — для всех runtime-операций (максимальная производительность).

## Стек технологий

**Backend:** Python 3.11, aiogram 3 (FSM), asyncpg, SQLAlchemy 2.0, Alembic (20 миграций), httpx, Pydantic, loguru

**Парсинг:** BeautifulSoup4 + lxml, Selenium + Chromium, fake-useragent

**Финтех:** T-Bank Invest SDK (gRPC), YooKassa SDK (54-ФЗ), cryptography (Fernet)

**Инфраструктура:** Docker Compose, PostgreSQL 15 (16 таблиц, JSONB, partial indexes), uv

## Ключевые достижения

- **Глубокая интеграция с российской финансовой инфраструктурой** — e-disclosure, НРД, АКРА, НРА, Эксперт РА, T-Bank, isin.ru
- **gRPC-интеграция T-Bank Invest API** — delta sync портфеля, определение эмитентов через asset_uid/borrow_name
- **Фискализация по 54-ФЗ** — полноценный приём платежей с электронными чеками через YooKassa
- **Event-driven дедупликация** — event_key предотвращает дублирование уведомлений
- **Fernet-шифрование** — API-токены T-Bank и контактные данные зашифрованы в БД
- **20 Alembic-миграций** — эволюция от простого бота до финтех-платформы
