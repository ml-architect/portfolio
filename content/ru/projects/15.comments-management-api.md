## О проекте

REST API для управления статьями, комментариями и авторами с полнотекстовым поиском. Гибридная архитектура: Elasticsearch для поиска + PostgreSQL для персистенции. 18 эндпоинтов, Docker Compose с 3 сервисами.

## Функционал

- CRUD для статей (10 эндпоинтов) и комментариев (6 эндпоинтов)
- Полнотекстовый поиск через Elasticsearch с мультиязычными анализаторами (russian + english)
- Импорт данных из Excel через pandas
- Агрегация авторов через ES terms aggregation
- Автодокументация Swagger UI

## Архитектура

Синхронная запись в оба хранилища — Elasticsearch (полнотекстовый поиск, морфологическая обработка) и PostgreSQL (реляционные запросы, индексы). Pydantic-settings для конфигурации (12-factor app). Graceful shutdown с корректным закрытием соединений.

## Стек технологий

Python, FastAPI, Uvicorn, Elasticsearch 8.9, PostgreSQL 16, Pydantic v2, pydantic-settings, pandas, Docker Compose
