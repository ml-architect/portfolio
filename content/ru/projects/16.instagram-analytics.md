## О проекте

Серия из 3 проектов, демонстрирующих эволюцию от простых скриптов к полноценному веб-приложению:

### anmi_web_app (2020)
Веб-приложение для аналитики Instagram-аккаунтов, развёрнутое на реальном хостинге anmi.space. Авторизация через сессии, парсинг через GraphQL API, анализ плотности публикаций по времени суток. Обход защиты: прокси, CSRF-токены, динамическое обновление заголовков.

### instagram_stat (2019)
Скрипт для автоматического сбора ежемесячной статистики через Instagram Private API — предшественник anmi_web_app.

### insta_bot (2019)
Бот для автоматизации с anti-detection стратегиями: рандомизированные задержки, мультиаккаунтный пул, stateful-конфигурация.

## Эволюция

**2019** — простые скрипты с готовыми библиотеками → **2020** — самостоятельный парсинг GraphQL, production-деплой.

## Стек технологий

Python 3, requests, BeautifulSoup (lxml), Instagram GraphQL API, InstagramAPI (Private API), configparser
