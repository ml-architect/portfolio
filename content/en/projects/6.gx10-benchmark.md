## About the Project

GX10 Benchmark is a tool for objective LLM performance benchmarking on GPUs, developed for a technical article publication on Habr. A reproducible methodology for evaluating new GPUs in the context of LLM inference.

The primary target GPU is the **NVIDIA GB10 (GX10)**, one of the first desktop chips on the Grace Blackwell architecture.

## Objective

Create a fully reproducible benchmarking tool that objectively measures and visualizes GPU performance during LLM inference across various scenarios, compares against reference GPUs, and generates ready-to-publish material for a technical article.

## Features

### Test Scenarios

- **Chatbot** — interactive chat: batch_size=1, contexts of 512–4096 tokens, TTFT measurement
- **Batch Processing** — bulk processing: batch_size 4–64, throughput
- **Long Context (RAG)** — up to 32,768 tokens, scaling evaluation
- **Stress Test** — sustained load with thermal throttling monitoring

### 13 Metrics

TTFT, TPOT, Throughput, GPU Memory, GPU Power, Tokens/Watt, Temperature, Precision, and others. Automatic bottleneck detection (compute-bound, memory-bound, thermal).

### Analytics

Comparison with reference GPUs (RTX 4090, A100, H100). FP16 vs FP8 with speedup calculation. Auto-generated Markdown report with recommendations.

## Tech Stack

**ML Inference:** vLLM, PyTorch 2.0+, Transformers 4.35+

**GPU Monitoring:** GPUtil, pynvml, py3nvml, psutil

**Data Analysis:** pandas, numpy, matplotlib, seaborn

**Infrastructure:** Docker (nvcr.io/nvidia/vllm), Docker Compose, Makefile (30+ commands)

## Key Achievements

- **Full Docker-ized reproducibility** — entire benchmark with a single `make quick` command, no manual setup
- **Thermal throttling detector** — comparison of first and second half of the test, threshold > 5% degradation
- **Energy efficiency** — tokens/Watt via pynvml, especially relevant for GB10
- **Production Makefile** — 30+ commands with colored output, confirmations, auto-detection of GPU/Docker
- **One of the first public NVIDIA GB10 benchmarks** — valuable material for the ML community
