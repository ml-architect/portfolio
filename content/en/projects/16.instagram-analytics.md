## About the Project

A series of 3 projects demonstrating the evolution from simple scripts to a full-fledged web application:

### anmi_web_app (2020)
Web application for Instagram account analytics, deployed on a real hosting at anmi.space. Session-based authorization, parsing via GraphQL API, publication density analysis by time of day. Protection bypass: proxies, CSRF tokens, dynamic header updates.

### instagram_stat (2019)
Script for automatic monthly statistics collection via Instagram Private API — predecessor of anmi_web_app.

### insta_bot (2019)
Automation bot with anti-detection strategies: randomized delays, multi-account pool, stateful configuration.

## Evolution

**2019** — simple scripts with off-the-shelf libraries → **2020** — independent GraphQL parsing, production deployment.

## Tech Stack

Python 3, requests, BeautifulSoup (lxml), Instagram GraphQL API, InstagramAPI (Private API), configparser
