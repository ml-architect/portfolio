## О проекте

Веб-приложение для распознавания лиц в реальном времени через веб-камеру браузера. WebRTC peer-to-peer стриминг видео с серверной обработкой.

## Задача

Реализовать real-time распознавание лиц, работающее прямо в браузере — через WebRTC стриминг видеопотока на сервер с ML-обработкой.

## Стек технологий

Python, Streamlit 1.23, streamlit-webrtc 0.45, aiortc 1.5, pandas 2.0, numpy 1.24, Pillow 9.5

## Ключевые достижения

- **WebRTC + ML** — технически сложная комбинация для real-time inference
- **Работа с видеопотоком** — не статичные изображения, а непрерывный стрим
- **Полноценное веб-приложение** — не jupyter-ноутбук
