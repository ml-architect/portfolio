## О проекте

«Цифровой прорыв» — всероссийская серия хакатонов по цифровизации госсектора: 48 часов на решение реальной задачи конкретного региона. Я участвовал в пяти проектах, охватывающих CV, NLP и RecSys.

## Проекты

### Выявление незаконных построек по спутниковым снимкам

**Регион:** Удмуртская Республика

Спутниковые снимки нарезались на тайлы и обрабатывались двумя пайплайнами: семантическая сегментация на ResNet-50 выделяла контуры зданий, а YOLOv5 детектировал объекты и переводил пиксельные координаты в географические. Найденные строения автоматически сверялись с кадастром Росреестра через API — незарегистрированные помечались как потенциальные нарушения.

### Классификация обращений граждан

**Регион:** Курская область

Автоматическая маршрутизация обращений граждан по ведомствам. Дообучены три модели семейства BERT на размеченном корпусе обращений: базовый BERT, ruBERT-tiny и Sentence-BERT. Модели определяли тематическую категорию текста и направляли обращение в нужный отдел.

### Классификация состояния коммунальной инфраструктуры

**Регион:** Республика Башкортостан

Автоматическое определение типа коммунального нарушения по фотографии. Применялось трансферное обучение Xception (fine-tuning верхних слоёв) и EfficientNet-B7 с аугментацией и динамическим learning rate. Для борьбы с дисбалансом классов использовалось взвешивание и очистка данных от ошибочных меток.

### Поиск музейных экспонатов

**Регион:** Липецкая область

Поиск похожих экспонатов среди 5 000+ предметов из разных музеев по изображению или текстовому описанию. Для каждого экспоната строились визуальные эмбеддинги (свёрточная сеть) и текстовые (по метаданным). Поиск ближайших соседей в пространстве эмбеддингов находил семантически близкие предметы без точного совпадения.

### Рекомендательная система для библиотеки

**Регион:** Тюменская область

Персональные рекомендации книг на основе истории выдач (~100 000 книг в каталоге). Гибридный подход: коллаборативная фильтрация через LightFM + контентные эмбеддинги книг на базе Transformers. Быстрый поиск ближайших соседей — через индекс FAISS. Финальное ранжирование уточнялось за 12 итераций.

## Стек технологий

**Computer Vision:** PyTorch, YOLOv5, segmentation-models-pytorch, EfficientNet-B7, Xception, OpenCV, albumentations

**NLP:** Hugging Face Transformers, BERT, ruBERT-tiny, Sentence-BERT

**RecSys:** LightFM, FAISS, CatBoost, TensorFlow/Keras

**Данные:** pandas, numpy, scipy, scikit-learn

- **5 кейсов за 48 часов каждый** — реальные задачи госсектора для конкретных регионов
- **CV + NLP + RecSys** — полный спектр ML-задач в одной серии соревнований
- **Интеграция CV с кадастром** — связка спутниковой сегментации с геоинформационными API Росреестра
- **Уникальные домены данных** — коммунальные нарушения, музейные коллекции, библиотечные фонды
