## О проекте

Коллекция из 5 решений хакатонов «Цифровой прорыв» — всероссийского конкурса по цифровизации регионов. Каждый проект решает реальную задачу для конкретного региона России.

## Проекты

### Выявление незаконных построек по спутниковым снимкам
**Регион:** Удмуртская Республика
Анализ спутниковых снимков для автоматического обнаружения незаконных построек. Computer Vision + Satellite imagery.

### Классификация обращений граждан
**Регион:** Курская область
Автоматическая классификация текстовых обращений граждан по категориям. NLP + Text classification.

### Оптимизация работы коммунальных служб
**Регион:** Республика Башкортостан
Оптимизация маршрутов и графиков коммунальных служб. Operations Research.

### Поиск музейных экспонатов
**Регион:** Липецк
Поиск похожих музейных экспонатов по изображению. Information Retrieval + Computer Vision.

### Рекомендательная система для библиотеки
**Регион:** Тюмень
Персональные рекомендации книг для посетителей библиотеки. RecSys.

## Ключевые достижения

- **5 реальных кейсов** для госсектора разных регионов
- **Широта компетенций** — CV, NLP, RecSys, Optimization в одном репозитории
- **Спутниковые снимки** — редкий и востребованный навык
- **Социально значимые задачи** — цифровизация регионов России
