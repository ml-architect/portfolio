## О проекте

Mediaclass API — бэкенд коммерческого десктопного приложения «МЕДИАКЛАСС ИИ» для автоматического распознавания речи и диаризации спикеров в аудио- и видеофайлах. Продукт для корпоративных клиентов — образовательных учреждений, медиакомпаний и организаций, которым нужно транскрибировать записи совещаний и лекций с идентификацией говорящих.

## Задача

Разработать API, который работает как локальный сервер с Electron-фронтендом и обеспечивает прозрачное переключение между локальным GPU-инференсом и облачным сервисом Palatine Cloud. Решение должно поставляться в защищённом виде с лицензированием по оборудованию.

## Архитектура

### Гибридные ML-провайдеры (Local + Cloud)

Ключевая архитектурная фишка — паттерн Provider на Protocol-based интерфейсах. Единый пайплайн транскрибации прозрачно работает как с локальными моделями (Whisper + Pyannote на GPU пользователя), так и с облачным Palatine Cloud API. Переключение — один параметр в запросе.

### ML-пайплайн диаризации

Production-grade диаризация: Pyannote.audio 4.0 с предобученными моделями сегментации, speaker embeddings и VBx-кластеризацией с PLDA. Модели поставляются с приложением.

### Отказоустойчивость

- Cloud Token Manager с fallback-цепочкой из 3 токенов
- HTTP retry с экспоненциальным backoff
- Автовалидация и восстановление повреждённых директорий из backup
- CPU Fallback при недоступности GPU

## Стек технологий

**Backend:** Python 3.10/3.11, FastAPI, Uvicorn, Pydantic v2, ORJSONResponse

**ML/AI:** OpenAI Whisper (whisper-timestamped), Pyannote.audio 4.0, PyTorch 2.8, torchaudio, scikit-learn, SciPy

**Медиа:** FFmpeg, Pydub, MoviePy, SoundFile

**Инфраструктура:** Docker, Conda, PyArmor (обфускация), Loguru

**Лицензирование:** WMI COM / PowerShell (Windows), machine-id (Linux), IOPlatformUUID (macOS)

## Ключевые достижения

- **Гибридная ML-архитектура** — Protocol-based провайдеры для бесшовного переключения между локальным и облачным инференсом
- **Промышленный пайплайн диаризации** — Pyannote.audio 4.0 с VBx-кластеризацией и PLDA
- **Коммерческая защита** — PyArmor шифрование + лицензирование по hardware hash (кроссплатформенное)
- **E2E тесты без GPU** — stub-классы для torch, numpy, moviepy позволяют тестировать без ML-зависимостей
- **Асинхронная обработка** — thread-safe TaskStore с отслеживанием прогресса в реальном времени
