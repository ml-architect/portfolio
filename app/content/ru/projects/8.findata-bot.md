## О проекте

Findata BOT — production-ready Telegram-бот для инвесторов на российском облигационном рынке. Агрегирует данные из шести источников — e-disclosure.ru, НРД, рейтинговые агентства АКРА, НРА и Эксперт РА, isin.ru — и доставляет персонализированные уведомления, отфильтрованные по ISIN-кодам облигаций в портфеле пользователя. Полностью монетизирован: тарифные планы, пробный период, приём платежей с фискализацией по 54-ФЗ.

## Задача

Превратить ежедневный ручной мониторинг шести разрозненных площадок в автоматическую доставку уведомлений. Интегрироваться с брокерским API для привязки портфеля, обеспечить безопасность финансовых данных и поддержать полный бизнес-цикл — от регистрации до оплаты.

## Функционал

### Мониторинг корпоративных раскрытий
Подписка на эмитентов с фильтрацией по ISIN-кодам и триггер-фразам. Отслеживаются четыре типа отчётности (устав, годовой отчёт, бухгалтерская отчётность, МСФО). Критичные события — неисполнение обязательств и т.п. — доставляются немедленно.

### Рейтинговые уведомления
Мониторинг АКРА, НРА, Эксперт РА и НРД. Два режима доставки: точечный — по совпадению ISIN из портфеля, и широковещательный — для событий, важных всем инвесторам.

### Мониторинг новых облигаций
Сбор свежих ISIN за скользящее окно 30 дней с проверкой доступности у брокера. Уведомление приходит, как только бумага становится доступна для покупки.

### Привязка брокерского портфеля
Интеграция с T-Bank Invest API: зашифрованное хранение токена, ночная синхронизация портфеля. На основе состава облигаций автоматически создаются, обновляются и деактивируются подписки на эмитентов.

### Монетизация
Подписочная модель: от бесплатного тарифа с лимитом компаний до премиального с полным доступом. Пробный период, напоминания об истечении, плавный даунгрейд без потери настроек.

## Архитектура

### Двухпроцессная модель
Первый процесс — Telegram-интерфейс (команды, меню, настройки). Второй — набор фоновых задач по расписанию: мониторинг источников, рассылка уведомлений, синхронизация портфелей, обработка платежей. Разделение гарантирует отзывчивость бота независимо от нагрузки на фон.

### Расширяемая система источников данных
Каждый источник — независимый модуль с единым интерфейсом: получение данных, парсинг, нормализация, доставка. Добавление нового сводится к реализации модуля и регистрации в реестре — планирование, дедупликация и доставка подхватываются автоматически.

### Управление подписками
При привязке портфеля подписки создаются автоматически по составу облигаций. При даунгрейде тарифа — мягко деактивируются без удаления. Ночная синхронизация обнаруживает изменения портфеля и корректирует подписки.

### Двухуровневый доступ к данным
Схема и миграции — через SQLAlchemy ORM + Alembic. Все runtime-операции — через asyncpg напрямую для максимальной производительности.

## Безопасность и надёжность

- **Шифрование** — API-токены и персональные данные хранятся в БД в зашифрованном виде (Fernet); при компрометации базы данные остаются защищены
- **Валидация** — строгая проверка всех пользовательских вводов (ссылки, ISIN, идентификаторы); admin-доступ по белому списку
- **Дедупликация** — предотвращение повторных уведомлений при сбоях, перезапусках и параллельной обработке
- **Graceful shutdown** — корректное завершение фоновых процессов без потери данных
- **Логирование** — структурированные журналы по компонентам с ротацией и компрессией (loguru)
- **Платежи** — YooKassa с автоматической фискализацией по 54-ФЗ

## Стек технологий

**Backend:** Python, aiogram 3 (FSM), asyncpg, SQLAlchemy, Alembic, httpx, Pydantic, loguru

**Парсинг:** BeautifulSoup4 + lxml, Selenium + Chromium, fake-useragent

**Финтех:** T-Bank Invest SDK (gRPC), YooKassa SDK (фискализация по 54-ФЗ), Fernet (шифрование)

**Инфраструктура:** Docker Compose, PostgreSQL (JSONB, partial indexes), uv

## Ключевые достижения

- **+10% доходности** личного облигационного портфеля за счёт своевременной реакции на рейтинговые события и раскрытия
- **Экономия нескольких часов в неделю** — автоматический мониторинг шести площадок вместо ручного, без пропущенных событий
- **Полный охват финансовой инфраструктуры РФ** — e-disclosure, НРД, АКРА, НРА, Эксперт РА, isin.ru, T-Bank Invest API
- **Автосинхронизация портфеля** — привязка брокерского счёта и автоматическое управление подписками
- **Полноценная монетизация** — приём платежей с фискализацией, управление жизненным циклом подписки
- **Расширяемая архитектура** — подключение новых источников без изменения существующего кода
