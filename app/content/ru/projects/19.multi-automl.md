## О проекте

Внутренний корпоративный инструмент — веб-платформа для автоматизированного обучения ML-моделей без написания кода. Построена на базе LightAutoML (разработка Сбера) и обёрнута в интерактивный интерфейс Streamlit. Позволяет аналитикам и бизнес-пользователям загружать табличные данные, выбирать тип задачи, настраивать параметры и получать обученную модель с визуализацией метрик — всё через браузер.

## Задача

Снизить порог входа в машинное обучение для сотрудников, не являющихся ML-инженерами. Вместо написания Python-кода и настройки пайплайнов пользователь загружает CSV, указывает целевую переменную и получает обученную модель с метриками одним нажатием кнопки.

## Архитектура

### Паттерн Multi-Model Orchestrator

Класс `MultiAutoML` принимает список `BaseModel`-экземпляров, последовательно обучает каждый с прогресс-баром, собирает предсказания и метрики. Позволяет сравнивать несколько конфигураций и масштабировать систему добавлением новых AutoML-бэкендов.

### Абстракция BaseModel

Унифицированный интерфейс (fit, predict, predict_proba, evaluate) поверх LightAutoML, скрывающий особенности API: отсутствие чистого `fit`, вывод вероятностей для бинарной классификации, OHE-формат для мультикласса.

### Три типа задач

**Бинарная классификация** — F1, Accuracy, ROC-AUC, Precision, Recall, PR-AUC. Визуализация: confusion matrix, ROC-AUC curve, PR curve.

**Мультиклассовая классификация** — F1, Accuracy, Precision, Recall (weighted). Валидация: минимум 5 записей на класс.

**Регрессия** — MAE, MSE, R2, MAPE, кастомная SMAPE.

## Стек технологий

**ML:** LightAutoML 0.3.7.3 (Сбер), scikit-learn 1.1.3, NumPy 1.23.5

**Веб-интерфейс:** Streamlit 1.15.1, Plotly 5.11.0

**Данные:** pandas 1.4.3

**Инфраструктура:** Docker, Docker Compose (порт 80)

## Ключевые достижения

- **Абстракция поверх LightAutoML** — `BaseModel` предоставляет scikit-learn-подобный интерфейс, нивелируя особенности API фреймворка
- **Расширяемая мультимодельная архитектура** — `MultiAutoML` позволяет добавлять новые AutoML-бэкенды без изменения UI
- **Русскоязычный интерфейс** — все элементы UI, ошибки и документация на русском для корпоративного использования
- **Встроенные демо-данные** — 3 скачиваемых датасета (Titanic, Body Fat, синтетический мультикласс) для быстрого знакомства
- **Кастомная метрика SMAPE** — реализация Symmetric Mean Absolute Percentage Error, отсутствующей в scikit-learn
- **Однокомандное развёртывание** — `docker-compose up -d` запускает готовое приложение
