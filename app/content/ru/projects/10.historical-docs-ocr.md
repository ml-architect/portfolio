## О проекте

Проект по распознаванию текста на исторических документах XVIII–XIX веков: губернаторские отчёты и уставные грамоты дореволюционной России. Работа с рукописными и печатными текстами из архива 29+ регионов.

## Задача

Построить пайплайн OCR, который может детектировать текстовые блоки на сканах старинных документов (часто повреждённых, с неровным почерком) и распознавать текст — как рукописный, так и печатный.

## Пайплайн

**Детекция** → **Сегментация** → **Распознавание**

- **YOLOv8** — детекция текстовых блоков на сканах
- **Roboflow** — профессиональная разметка данных
- **Hugging Face Transformers** — seq2seq модели для OCR
- **ClearML** — трекинг экспериментов и версионирование

## Стек технологий

**ML:** PyTorch 2.2 (CUDA 11.8), Ultralytics (YOLOv8), Hugging Face Transformers 4.37, Datasets, scikit-learn

**CV:** OpenCV, Roboflow, Supervision

**MLOps:** ClearML, JupyterLab 4.1

## Ключевые достижения

- **Уникальный домен** — OCR дореволюционных рукописных документов, нетривиальная задача для CV/NLP
- **Object Detection + seq2seq** — комбинация YOLOv8 для детекции и Transformers для распознавания
- **MLOps-практики** — ClearML для трекинга, Roboflow для разметки, версионирование данных
- **Реальный архив** — 29+ регионов, губернаторские отчёты 1804-1882 годов
