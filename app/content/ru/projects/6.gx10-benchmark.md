## О проекте

GX10 Benchmark — инструмент для объективного бенчмаркинга производительности LLM на GPU, разработанный для публикации технической статьи на Хабр. Воспроизводимая методология для оценки новых GPU в контексте LLM-инференса.

Основной целевой GPU — **NVIDIA GB10 (GX10)**, один из первых десктопных чипов на архитектуре Grace Blackwell.

## Задача

Создать полностью воспроизводимый инструмент бенчмаркинга, который объективно измеряет и визуализирует производительность GPU при инференсе LLM в различных сценариях, сравнивает с референсными GPU и генерирует готовый материал для технической статьи.

## Функционал

### Сценарии тестирования

- **Chatbot** — интерактивный чат: batch_size=1, контексты 512-4096 токенов, измерение TTFT
- **Batch Processing** — массовая обработка: batch_size 4-64, пропускная способность
- **Long Context (RAG)** — до 32768 токенов, оценка масштабирования
- **Stress Test** — длительная нагрузка с мониторингом thermal throttling

### 13 метрик

TTFT, TPOT, Throughput, GPU Memory, GPU Power, Tokens/Watt, Temperature, Precision и другие. Автоматическое определение bottleneck (compute-bound, memory-bound, thermal).

### Аналитика

Сравнение с референсными GPU (RTX 4090, A100, H100). FP16 vs FP8 с расчётом speedup. Автогенерация Markdown-отчёта с рекомендациями.

## Стек технологий

**ML Inference:** vLLM, PyTorch 2.0+, Transformers 4.35+

**GPU Monitoring:** GPUtil, pynvml, py3nvml, psutil

**Data Analysis:** pandas, numpy, matplotlib, seaborn

**Инфраструктура:** Docker (nvcr.io/nvidia/vllm), Docker Compose, Makefile (30+ команд)

## Ключевые достижения

- **Полная Docker-ized воспроизводимость** — весь бенчмарк одной командой `make quick`, никакой ручной настройки
- **Детектор thermal throttling** — сравнение первой и второй половины теста, порог > 5% деградации
- **Энергоэффективность** — tokens/Watt через pynvml, особенно актуально для GB10
- **Production Makefile** — 30+ команд с цветным выводом, подтверждениями, автопроверкой GPU/Docker
- **Один из первых публичных бенчмарков NVIDIA GB10** — ценный материал для ML-сообщества
