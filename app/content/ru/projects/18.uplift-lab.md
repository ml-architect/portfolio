## О проекте

Интерактивное веб-приложение для демонстрации uplift-моделирования в маркетинговой аналитике, разработанное для Лаборатории интеллектуального анализа РАНХиГС. Реализует концепцию «Аналитик vs ML» — пользователь вручную сегментирует аудиторию через систему фильтров, затем видит delta-метрики, наглядно показывающие превосходство ML-моделей.

Используется классический бенчмарк-датасет MineThatData E-Mail Analytics (Kevin Hillstrom) — 42 693 записи клиентов email-маркетинговой кампании.

## Задача

Создать образовательный инструмент, демонстрирующий преимущества uplift-моделирования перед ручной сегментацией аудитории. Uplift-моделирование определяет, на каких клиентов рекламное воздействие окажет максимальный эффект, позволяя оптимизировать маркетинговый бюджет.

## Архитектура

### ML-пайплайн

12 комбинаций моделей: 3 алгоритма (CatBoost, XGBoost, Random Forest) × 4 uplift-подхода (SoloModel, TwoModels vanilla, TwoModels DDR-control, ClassTransformation) через библиотеку scikit-uplift.

Лучший результат: **CatBoost SoloModel** — uplift@30% = 0.0776.

Предсказания предсохранены в CSV для мгновенной загрузки без GPU-зависимости. Фиксированные индексы train/test split для полной воспроизводимости.

### Интерактивная фильтрация

6 типов фильтров: канал покупки, тип клиента, тип товаров, ценовые сегменты (7 чекбоксов), почтовый индекс, recency (слайдер). Валидация на каждом шаге с информативными сообщениями об ошибках.

### CI/CD

GitHub Actions: проверка размера файлов + автодеплой на Hugging Face Spaces. Git LFS для бинарных артефактов.

## Стек технологий

**ML:** CatBoost 1.0.6 (GPU), XGBoost 1.6.1 (GPU), scikit-learn 1.1.1, scikit-uplift 0.4.1, category-encoders 2.5.0

**Веб-интерфейс:** Streamlit 1.11.0, Plotly 5.9.0, Matplotlib 3.5.2

**Данные:** pandas 1.4.3, NumPy 1.23.1, SciPy 1.8.1

**Инфраструктура:** GitHub Actions, Hugging Face Spaces, Git LFS

## Ключевые достижения

- **12 комбинаций моделей** — систематическое сравнение 3 алгоритмов в 4 подходах к uplift-моделированию
- **Концепция «Аналитик vs ML»** — пользователь сначала сегментирует вручную, затем видит delta-метрики превосходства ML
- **Специализированные uplift-метрики** — Qini AUC, uplift@k, weighted average uplift с визуализацией Qini/Uplift curves
- **Визуализация с пользовательской точкой** — красный маркер на ML-кривых показывает позицию ручной аналитики
- **GPU-ускорение обучения** — CatBoost (task_type="GPU") и XGBoost (tree_method='gpu_hist')
