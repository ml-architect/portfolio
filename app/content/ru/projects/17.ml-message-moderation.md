## О проекте

Интерактивная лабораторная работа для модерации текстовых сообщений, разработанная для Лаборатории интеллектуального анализа РАНХиГС. Приложение позволяет в реальном времени тестировать и сравнивать три подхода к выявлению скрытых контактных данных в сообщениях пользователей торговых площадок: rule-based (Regex), классический ML (TF-IDF + CatBoost) и deep learning (BERT + CatBoost).

Помимо интерактивного тестирования моделей, приложение включает образовательный компонент: теоретические блоки с описанием каждого алгоритма и интерактивный квиз из 8 вопросов для проверки усвоения материала.

## Задача

Создать образовательно-исследовательский инструмент, демонстрирующий эволюцию подходов к модерации контента: от простых регулярных выражений до нейросетевых моделей. Бизнес-задача — выявление замаскированных контактных данных (замена символов цифрами, вставка пробелов, запись чисел словами) для предотвращения обхода комиссии площадки.

## Архитектура

### Три уровня классификации

**Regex** — детекция URL (HTTP/HTTPS, IPv4, IPv6, домены), email и телефонов через комплексные паттерны.

**TF-IDF + CatBoost** — пайплайн NLP-предобработки (удаление спецсимволов, пунктуации, нормализация, лемматизация через NLTK) → TfidfVectorizer → CatBoost-классификатор.

**BERT + CatBoost** — токенизация через DeepPavlov/rubert-base-cased-conversational → padding до 512 токенов → извлечение CLS-эмбеддинга (torch.no_grad) → CatBoost-классификатор. Подход feature extraction вместо fine-tuning — снижает требования к ресурсам.

### CI/CD

GitHub Actions: проверка размера файлов (10 MB лимит) + автоматическая синхронизация с Hugging Face Spaces. Git LFS для хранения моделей и данных.

## Стек технологий

**ML/AI:** PyTorch 1.12.1, Hugging Face Transformers 4.21.1, DeepPavlov/rubert-base-cased-conversational, CatBoost 1.0.6, scikit-learn 1.1.2, NLTK 3.7

**Веб-интерфейс:** Streamlit 1.12.0, Plotly 5.10.0

**Инфраструктура:** GitHub Actions, Hugging Face Spaces, Git LFS

## Ключевые достижения

- **Сравнительный анализ трёх подходов** — Regex, TF-IDF + CatBoost, BERT + CatBoost на одной задаче с наглядной демонстрацией сильных и слабых сторон
- **Русскоязычная NLP-модель** — DeepPavlov/rubert-base-cased-conversational, оптимальная для модерации русскоязычных сообщений
- **CatBoost как классификатор поверх BERT-эмбеддингов** — feature extraction вместо fine-tuning, значительное снижение вычислительных требований
- **Образовательный цикл** — теория + практика + квиз в одном приложении
- **Автоматический CI/CD** — push в main → синхронизация с Hugging Face Spaces
