## О проекте

REST-сервис для автоматического обнаружения сдвига ракурса камеры. Принимает два изображения — эталонное и проверяемое — и определяет, изменилось ли положение камеры по осям X (панорамирование), Y (наклон) и масштабу (зум).

Решает задачу мониторинга фиксированных камер видеонаблюдения: фиксирует несанкционированное перемещение или поворот камеры и сообщает направление сдвига.

## Задача

Разработать API-сервис для надёжного определения изменений ракурса камеры с фильтрацией ложных срабатываний от движущихся объектов (люди, транспорт). Сервис должен работать автономно, поддерживать GPU-ускорение и контейнеризированный деплой.

## Архитектура

### Пайплайн обработки

Входные изображения проходят предобработку: конвертация из байтового потока в NumPy-массив, ресайз до 1497×843, перевод в оттенки серого и гауссово сглаживание (ядро 5×5) для подавления шума.

### Детекция сдвига через оптический поток

Ядро системы — алгоритм **Farnebäck** для вычисления плотного оптического потока между парой изображений. По средним смещениям классифицируется направление сдвига с пороговой фильтрацией.

### Маскировка движущихся объектов

Для исключения ложных срабатываний — маскировка через **YOLOv8x-seg**. Модель детектирует людей, автомобили, мотоциклы, автобусы, грузовики и заменяет области масок медианным цветом. Оптический поток вычисляется только по статичным элементам сцены.

### REST API

FastAPI с тремя эндпоинтами: health check (`GET /health`), детекция сдвига (`POST /detect_shift`) и Swagger UI (`GET /doc`). Эндпоинт принимает два изображения через multipart/form-data и опциональный флаг исключения движущихся объектов.

## Стек технологий

**Backend:** Python 3.10, FastAPI 0.100, Uvicorn, Pydantic

**Computer Vision:** OpenCV 4.8 (Farnebäck optical flow, Gaussian blur), Pillow

**ML:** PyTorch 2.0.1 (CUDA 11.7), Ultralytics YOLOv8x-seg

**Инфраструктура:** Docker, Docker Compose, NVIDIA Container Runtime

## Ключевые достижения

- **Автоматизация мониторинга** — замена ручного контроля положения камер
- **Снижение ложных срабатываний на ~85%** — маскировка движущихся объектов через YOLOv8x-seg
- **Обработка менее 1 секунды** — пригодно для периодического мониторинга в реальном времени
- **Обнаружение минимальных сдвигов** — пороговая фильтрация выявляет незначительные смещения камеры
- **Простая интеграция** — REST API со Swagger UI, подключение без изменения инфраструктуры
