## О проекте

Palatine Cloud — коммерческая облачная платформа, спроектированная и разработанная мной с нуля. Предоставляет бизнесу API для полного цикла обработки речи: ASR, диаризация, анализ тональности, суммаризация через локальные LLM и синтез речи (TTS).

Все ML-модели, включая LLM, работают локально на собственных GPU-серверах — это обеспечивает конфиденциальность данных, предсказуемые затраты и контроль над качеством инференса. Платформа обрабатывает **10 млн аудиоминут/день**, поддерживает **100+ языков** с автодетекцией, принимает **23+ аудио/видео формата** и обеспечивает **uptime 99.982%** на 4 географически распределённых дата-центрах Tier III.

Проект полностью монетизирован: тарифы, приём платежей, автоматическая фискализация — платформа приносит стабильный доход.

## Задача

Создать коммерческую облачную платформу для обработки больших объёмов речевых данных в реальном времени: масштабируемую, надёжную, с высоким качеством распознавания, OpenAI-совместимым API, локальными LLM и полным бизнес-циклом от регистрации до оплаты и фискализации.

## Архитектура

Платформа построена на **микросервисной архитектуре из 10 сервисов** на нескольких выделенных GPU-серверах. Взаимодействие — HTTP и gRPC, единый API Gateway, полный стек мониторинга.

### Распознавание речи и диаризация
Двухуровневая архитектура: оркестрация (аутентификация, биллинг, маршрутизация) и ML-инференс. Модели обслуживаются через **NVIDIA Triton Inference Server** (gRPC, sequence batching, stateful inference), оркестрация GPU-ресурсов — через **Ray Serve** с автоскейлингом реплик.

Стриминговая транскрипция: клиент отправляет аудиопоток через WebSocket, сервис декодирует и ресемплирует сигнал, передаёт на Triton через gRPC и возвращает результат по мере распознавания. Поддержка 15+ кодеков (PCM, Opus, FLAC, MP3, AAC и др.), частоты 8–48 кГц. **Точность 92.9% (WER 7.1%)**, файлы до 10+ часов, латентность ~25 с на 30-минутную запись.

### Локальные LLM — суммаризация и аналитика
**Локально развёрнутые LLM** для обработки распознанного текста. Библиотека настраиваемых промптов, режим thinking для сложных задач. OpenAI-совместимый эндпоинт chat completions — клиенты используют стандартный OpenAI SDK. Все данные остаются на серверах платформы.

### API Gateway
Единая точка входа: декларативная маршрутизация с динамической загрузкой, несколько типов аутентификации, проксирование ко внутренним сервисам, составные запросы — цепочки вызовов с подстановкой промежуточных результатов. Фронтирован reverse proxy с TLS-терминацией и автоматическими сертификатами.

### Платёжный шлюз
Построен по **Clean Architecture**: сущности, бизнес-логика, интерфейсы, адаптеры. Мультимерчант-маршрутизация с учётом тарифа, обработка webhook с проверкой подлинности, фискальные чеки по 54-ФЗ, промокоды, идемпотентная обработка событий.

### Self-hosted Supabase
Полный стек из 12+ контейнеров: аутентификация, real-time подписки, хранилище файлов, серверлесс-функции, REST API, пулинг соединений. Кастомные SQL-миграции с политиками RLS.

### Мониторинг и наблюдаемость
Мультинодная архитектура: дашборды и визуализация, сбор метрик со всех серверов (включая Ray Serve и GPU), централизованные логи от всех Docker-контейнеров, агенты сбора на удалённых хостах.

### Синтез речи (TTS)
Контейнеризированный нейросетевой TTS с GPU-инференсом, клонированием голоса по референсному аудио и OpenAI-совместимым API.

### Личный кабинет, админка и документация
Бэкенд личного кабинета с rate limiting, админ-панель управления тарифами и промокодами, **публичная API-документация** на Mintlify — автогенерация из OpenAPI-спецификаций с интерактивным Playground.

## Стек технологий

**Backend:** Python 3.10–3.11, FastAPI, Uvicorn, Pydantic, httpx, WebSocket

**ML/AI:** NVIDIA Triton Inference Server (gRPC, sequence batching), Ray Serve (автоскейлинг), CUDA 12.6, Fish Speech (TTS), локальные LLM (Ollama), CTC Decoder

**Данные:** PostgreSQL 15, Supabase (self-hosted), Redis, Alembic

**Инфраструктура:** Docker / Docker Compose, GitLab CI/CD (lint → test → deploy), Traefik (TLS, HSTS), Grafana, Prometheus, Loki, Alloy

**Интеграции:** YooKassa (54-ФЗ), Supabase Auth (JWT), OpenAI-совместимый API, Mintlify (автогенерация документации)
