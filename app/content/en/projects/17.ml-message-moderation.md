## About the Project

An interactive lab for text message moderation, developed for the Intelligent Analysis Laboratory at RANEPA. The application allows real-time testing and comparison of three approaches to detecting hidden contact information in marketplace user messages: rule-based (Regex), classical ML (TF-IDF + CatBoost), and deep learning (BERT + CatBoost).

In addition to interactive model testing, the application includes an educational component: detailed theory sections describing each algorithm and an interactive quiz of 8 questions to test comprehension.

## Objective

Build an educational and research tool demonstrating the evolution of content moderation approaches: from simple regular expressions to neural network models. The business problem is detecting disguised contact information (replacing characters with digits, inserting spaces, writing numbers as words) to prevent marketplace commission circumvention.

## Architecture

### Three Classification Levels

**Regex** — URL detection (HTTP/HTTPS, IPv4, IPv6, domains), email, and phone numbers via complex patterns.

**TF-IDF + CatBoost** — NLP preprocessing pipeline (special character removal, punctuation removal, normalization, NLTK lemmatization) → TfidfVectorizer → CatBoost classifier.

**BERT + CatBoost** — tokenization via DeepPavlov/rubert-base-cased-conversational → padding to 512 tokens → CLS embedding extraction (torch.no_grad) → CatBoost classifier. Feature extraction approach instead of fine-tuning — reduces resource requirements.

### CI/CD

GitHub Actions: file size check (10 MB limit) + automatic sync with Hugging Face Spaces. Git LFS for model and data storage.

## Tech Stack

**ML/AI:** PyTorch 1.12.1, Hugging Face Transformers 4.21.1, DeepPavlov/rubert-base-cased-conversational, CatBoost 1.0.6, scikit-learn 1.1.2, NLTK 3.7

**Web Interface:** Streamlit 1.12.0, Plotly 5.10.0

**Infrastructure:** GitHub Actions, Hugging Face Spaces, Git LFS

## Key Achievements

- **Comparative analysis of three approaches** — Regex, TF-IDF + CatBoost, BERT + CatBoost on the same task with clear demonstration of strengths and weaknesses
- **Russian-language NLP model** — DeepPavlov/rubert-base-cased-conversational, optimal for Russian message moderation
- **CatBoost as classifier on top of BERT embeddings** — feature extraction instead of fine-tuning, significantly reduced compute requirements
- **Complete educational cycle** — theory + practice + quiz in one application
- **Automated CI/CD** — push to main → sync with Hugging Face Spaces
