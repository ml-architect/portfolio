## About the Project

An interactive lab exercise for the [RANEPA Intelligent Analysis Laboratory](https://lia.ranepa.ru/), integrated into the educational platform. Students study NLP methods on a real-world task — detecting disguised contact information in marketplace messages (letter-to-digit substitution, spaces within words, numbers written as words).

The application lets users enter arbitrary messages and instantly see the verdict of three algorithms — from regex to BERT. Users can load a random message from a real dataset. Each algorithm is accompanied by theory with formulas, and at the end there is a quiz of 8 questions (passing threshold: 7/8). A complete "theory → practice → knowledge check" cycle.

## Objective

Develop an interactive tool demonstrating the evolution of content moderation approaches — from manual rules to neural networks — on a real business task, accessible to students without deep ML experience.

## Architecture

The web interface handles UX and educational content, while ML logic is extracted into a separate module. The core consists of three independent classifiers of increasing complexity:

**Level 1 — Regex.** Pattern matching for URLs (including IPv4/IPv6), emails, and phone numbers in any notation (spaces, hyphens, parentheses). Reliable for explicit contacts, powerless against disguise.

**Level 2 — TF-IDF + CatBoost.** Text preprocessing (cleaning, normalization, lemmatization), followed by TF-IDF vectorization — evaluating word importance relative to the corpus. The resulting vector is fed into a CatBoost classifier.

**Level 3 — BERT + CatBoost.** Russian-language model DeepPavlov/rubert-base-cased-conversational (trained on conversational texts). The message is tokenized, passed through the neural network, and the resulting embedding is fed into CatBoost. Feature extraction instead of full fine-tuning — dramatically reduces computational requirements without significant quality loss.

The three-level architecture clearly demonstrates how recognition quality improves as algorithm complexity increases.

### Deployment and CI/CD

Deployed on Hugging Face Spaces and integrated into the RANEPA Intelligent Analysis Laboratory platform. Models and datasets are stored via Git LFS. CI/CD: GitHub Actions checks file sizes and synchronizes the repository with HF Spaces on each push.

## Tech Stack

**ML/AI:** PyTorch, Hugging Face Transformers, DeepPavlov/rubert-base-cased-conversational, CatBoost, scikit-learn, NLTK

**Web Interface:** Streamlit, Plotly

**Infrastructure:** GitHub Actions, Hugging Face Spaces, Git LFS

## Key Achievements

- **Three levels of analysis** — Regex → TF-IDF + CatBoost → BERT + CatBoost with a clear comparison of strengths and weaknesses
- **Russian-language BERT** — DeepPavlov/rubert-base-cased-conversational, optimal for moderating conversational messages
- **Feature extraction instead of fine-tuning** — CatBoost on top of BERT embeddings, dramatically reducing computational requirements
- **Educational cycle** — theory with formulas, interactive practice, quiz (7/8) in a single application
- **CI/CD** — GitHub Actions → Hugging Face Spaces
