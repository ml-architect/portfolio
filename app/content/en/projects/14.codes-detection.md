## About the Project

A REST API service for detecting and decoding QR codes in images. A containerized production-ready microservice.

## Features

- POST `/decode_qr` — accepts base64-encoded images
- QR code detection and decoding via OpenCV + pyzbar
- Structured JSON responses with error handling

## Tech Stack

Python 3.10, FastAPI 0.78, OpenCV 4.5, pyzbar 0.1.9, uvicorn, Docker
