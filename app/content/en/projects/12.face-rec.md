## About the Project

A web application for real-time face recognition through a browser webcam. WebRTC peer-to-peer video streaming with server-side processing.

## Objective

Implement real-time face recognition that works directly in the browser — via WebRTC video stream to the server with ML processing.

## Tech Stack

Python, Streamlit 1.23, streamlit-webrtc 0.45, aiortc 1.5, pandas 2.0, numpy 1.24, Pillow 9.5

## Key Achievements

- **WebRTC + ML** — a technically challenging combination for real-time inference
- **Video stream processing** — not static images, but a continuous stream
- **Full-fledged web application** — not a Jupyter notebook
