## About the Project

A/B-test Lab is an interactive web application for conducting and analyzing A/B tests. Not a notebook, but a full-fledged tool for Data Science teams. Deployed on Hugging Face Spaces.

## Features

- Sample size and statistical power calculation
- P-value computation with interactive visualization
- Support for various statistical tests
- Interactive charts powered by Plotly

## Tech Stack

Python 3.9, Streamlit 1.10, scipy 1.8, statsmodels 0.13, plotly 5.9, altair 4.2, pandas 1.4, numpy 1.23
