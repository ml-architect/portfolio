## About the Project

A REST API for managing articles, comments, and authors with full-text search. Hybrid architecture: Elasticsearch for search + PostgreSQL for persistence. 18 endpoints, Docker Compose with 3 services.

## Features

- CRUD for articles (10 endpoints) and comments (6 endpoints)
- Full-text search via Elasticsearch with multilingual analyzers (Russian + English)
- Data import from Excel via pandas
- Author aggregation via ES terms aggregation
- Auto-generated Swagger UI documentation

## Architecture

Synchronous writes to both stores — Elasticsearch (full-text search, morphological processing) and PostgreSQL (relational queries, indexes). Pydantic-settings for configuration (12-factor app). Graceful shutdown with proper connection cleanup.

## Tech Stack

Python, FastAPI, Uvicorn, Elasticsearch 8.9, PostgreSQL 16, Pydantic v2, pydantic-settings, pandas, Docker Compose
