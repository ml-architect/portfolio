## About the Project

An interactive web application for demonstrating uplift modeling in marketing analytics, developed for the Intelligent Analysis Laboratory at RANEPA. Implements the "Analyst vs ML" concept — users manually segment the audience through a filter system, then see delta metrics clearly showing the superiority of ML models.

Uses the classic MineThatData E-Mail Analytics benchmark dataset (Kevin Hillstrom) — 42,693 customer records from an email marketing campaign.

## Objective

Create an educational tool demonstrating the advantages of uplift modeling over manual audience segmentation. Uplift modeling identifies which customers will respond most positively to marketing intervention, enabling marketing budget optimization.

## Architecture

### ML Pipeline

12 model combinations: 3 algorithms (CatBoost, XGBoost, Random Forest) × 4 uplift approaches (SoloModel, TwoModels vanilla, TwoModels DDR-control, ClassTransformation) via the scikit-uplift library.

Best result: **CatBoost SoloModel** — uplift@30% = 0.0776.

Predictions are pre-saved in CSV files for instant loading without GPU dependency. Fixed train/test split indices for full reproducibility.

### Interactive Filtering

6 filter types: purchase channel, customer type, product type, price segments (7 checkboxes), zip code, recency (slider). Validation at each step with informative error messages.

### CI/CD

GitHub Actions: file size check + auto-deploy to Hugging Face Spaces. Git LFS for binary artifacts.

## Tech Stack

**ML:** CatBoost 1.0.6 (GPU), XGBoost 1.6.1 (GPU), scikit-learn 1.1.1, scikit-uplift 0.4.1, category-encoders 2.5.0

**Web Interface:** Streamlit 1.11.0, Plotly 5.9.0, Matplotlib 3.5.2

**Data:** pandas 1.4.3, NumPy 1.23.1, SciPy 1.8.1

**Infrastructure:** GitHub Actions, Hugging Face Spaces, Git LFS

## Key Achievements

- **12 model combinations** — systematic comparison of 3 algorithms across 4 uplift modeling approaches
- **"Analyst vs ML" concept** — users segment manually first, then see delta metrics showing ML superiority
- **Specialized uplift metrics** — Qini AUC, uplift@k, weighted average uplift with Qini/Uplift curve visualization
- **User point visualization** — red marker on ML curves shows the manual analytics position
- **GPU-accelerated training** — CatBoost (task_type="GPU") and XGBoost (tree_method='gpu_hist')
