## About the Project

An analytical web application for predicting consumer activity using ML models and geospatial analytics. Fully containerized.

## Features

- ML models (CatBoost, scikit-learn) for classification and regression
- Geospatial analytics — activity mapped to locations via geopy
- Interactive dashboards powered by Plotly

## Tech Stack

Python, Streamlit 1.15, CatBoost 1.1, scikit-learn 1.2, plotly 5.11, geopy 2.3, pandas 1.5, Docker Compose
