## Быстрый старт

### Требования

- Node.js 18+
- pnpm (или npm)

### Установка и запуск

```bash
# Установка зависимостей
pnpm install

# Запуск dev-сервера
pnpm dev

# Сборка для продакшена (SSG)
pnpm generate

# Превью сборки
pnpm preview
```

### Добавление контента

- **Новый проект** — см. [projects.md](projects.md)
- **Новая статья блога** — см. [blog.md](blog.md)

Контент хранится в `app/content/` и разделён по локалям:

```
app/content/
├── ru/
│   ├── blog/        # Статьи блога (русский)
│   └── projects/    # Проекты (русский)
└── en/
    ├── blog/        # Статьи блога (английский)
    └── projects/    # Проекты (английский)
```

Каждый файл контента должен существовать в обеих локалях (ru и en).
