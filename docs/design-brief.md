# Design Brief — Portfolio

## Общее описание

Персональный сайт-портфолио ML-архитектора. Тёмная тема (единственная, без переключателя). Визуальный стиль — минималистичный, технологичный, с акцентом на типографику и пространство. Все страницы генерируются статически (SSG) для GitHub Pages.

## Визуальные эффекты (CSS)

### Gradient text
```css
background: linear-gradient(135deg, #8580ff 0%, #6C63FF 50%, #a3a1ff 100%);
```
Применяется через класс `.gradient-text`.

### Shimmer border
Вращающийся conic-gradient бордер (`conic-gradient(from var(--angle), transparent 70%, #ffffff 100%)`). Период: 4s, линейная анимация.

### Grid pattern
Сетка 60×60px из полупрозрачных линий (`rgba(255,255,255, 0.03)`).

### Noise overlay
SVG fractalNoise текстура через `::before` с opacity 0.03.

### Marquee
Бесконечная горизонтальная прокрутка (`translateX(-50%)`), 50s, линейная.

## CSS-анимации (Tailwind keyframes)

| Название      | Тип            | Длительность | Easing                      | Описание                    |
|---------------|----------------|--------------|-----------------------------|-----------------------------|
| `shimmer`     | opacity пульс  | 2s           | ease-in-out                 | 0.5 → 1 → 0.5              |
| `float`       | translateY     | 6s           | ease-in-out                 | 0 → -20px → 0              |
| `pulse-slow`  | pulse          | 4s           | cubic-bezier(0.4,0,0.6,1)  | Стандартный pulse, медленный|
| `spin-slow`   | rotation       | 4s           | linear                      | Полный оборот 360°          |

## Переходы страниц (Vue)

```
opacity 0.3s ease + transform 0.3s ease
Enter: opacity 0 → 1, translateY(10px → 0)
Leave: opacity 1 → 0, translateY(0 → -10px)
```

## GSAP-анимации

### Зарегистрированные плагины

| Плагин          | Назначение                                    |
|-----------------|-----------------------------------------------|
| **ScrollTrigger** | Запуск анимаций при скролле                  |
| **SplitText**     | Разбивка текста на символы/слова для анимации|
| **Flip**          | FLIP-анимации при изменении layout           |

### Глобальные настройки GSAP

```ts
gsap.defaults({
  ease: 'power2.out',
  duration: 0.8,
})
```

### Основной паттерн — Blur-In (размытие с появлением)

Главный и наиболее часто используемый паттерн анимации на сайте. Элемент плавно появляется снизу, одновременно теряя размытие и набирая непрозрачность.

```
FROM: { y: 20–40px, opacity: 0, filter: blur(10–20px) }
  TO: { y: 0, opacity: 1, filter: blur(0px) }
```

#### Пресеты Blur-In (composable `useGsap`)

| Вариант   | y     | blur   | duration | scrollTrigger start |
|-----------|-------|--------|----------|---------------------|
| `header`  | 30px  | 12px   | 0.9s     | top 80%             |
| `card`    | 40px  | 16px   | 0.8s     | top 85%             |
| `small`   | 20px  | 10px   | 0.7s     | top 85%             |

### Анимации по компонентам

#### HeroSection — Главный экран
- **SplitText** — разбивка заголовка на символы
- **Timeline** — последовательная цепочка анимаций:
  1. Строка 1: символы появляются с `blur(20px)`, stagger `0.03s`, delay `0.1s`
  2. Строка 2: аналогично, delay `0.3s`
  3. Scroll indicator: fade-in снизу, relative delay `-0.3s`
  4. Hero bottom: slide-up с blur, relative delay `-0.5s`
- Easing: `power3.out` (единственное место с power3)

#### FeaturedProjects — Избранные проекты
- Заголовок секции: blur-in `header`
- Карточки: blur-in `card`, stagger `0.12s`

#### FeaturedProjectCard — 3D-эффект при наведении
- **Mouse-driven 3D rotation**: `rotationX`, `rotationY` (±4°)
- `transformPerspective: 800`
- Duration: 0.3s (hover), 0.5s (leave — возврат в исходное)
- Отключается на touch-устройствах

#### StatsSection — Секция статистики
- **CountUp**: числовой счётчик от 0 до целевого значения
- Duration: 2s, ScrollTrigger start: `top 85%`

#### SkillsSection — Навыки
- Заголовок: blur-in `header`
- Элементы: blur-in `card`, stagger `0.12s`

#### ImpactSection — Секция достижений
- Заголовок: blur-in `header`, start: `top 90%`
- Карточки: blur-in `card`, stagger `0.1s`
- Счётчики: CountUp, duration `4s`, каскадный delay `index × 0.2s`

#### MarqueeSection — Бегущая строка
- Простой fade-in: `opacity: 0 → 1`, duration `1s`

#### ContactSection — CTA-секция
- Заголовок: blur-in `card` (blur 16px)
- Кнопка: blur-in `small` (blur 10px)

#### ProjectGrid — Сетка проектов
- Карточки: blur-in `card`, stagger `0.08s`
- `overwrite: true` для предотвращения конфликтов при фильтрации

#### ScrollReveal — Переиспользуемый компонент
- 4 пресета: `fadeUp`, `fadeIn`, `slideLeft`, `slideRight`
- Slide-анимации используют `x: ±40px`

#### AnimatedText — Анимированный текст
- Посимвольная анимация: stagger `0.02s`
- Пословная анимация: stagger `0.06s`
- `gsap.from()` с `y: 20, opacity: 0`

### Страницы с анимациями

| Страница            | Анимации                                           |
|---------------------|----------------------------------------------------|
| `index.vue`         | Все секции главной (Hero, Stats, Skills и т.д.)    |
| `about.vue`         | Blur-in для секций экспертизы, навыков, различий   |
| `projects/index.vue`| Blur-in заголовка и фильтров                       |
| `projects/[slug].vue`| Blur-in секций проекта (без ScrollTrigger)        |
| `blog/index.vue`    | Blur-in заголовка и карточек                       |
| `blog/[slug].vue`   | Blur-in контента статьи                            |

### Easing-функции

| Easing         | Использование                                 |
|----------------|-----------------------------------------------|
| `power2.out`   | ~90% анимаций — основной easing               |
| `power3.out`   | HeroSection timeline (символы заголовка)       |

### ScrollTrigger — конфигурация

- **start**: от `top 80%` (заголовки) до `top 95%` (карточки, about)
- **toggleActions**: `play none none none` — однократное воспроизведение
- Анимации запускаются лениво при скролле, что экономит ресурсы

### Stagger-значения

| Контекст    | Stagger  |
|-------------|----------|
| Символы     | 0.02–0.03s |
| Слова       | 0.06s      |
| Карточки    | 0.08–0.12s |
